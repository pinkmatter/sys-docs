# FarEarth 3.19.1 release notes
#### *Release Date - 9 May 2025*

> Back to the [All Release notes](releaseNotesSummary.md)

### New Features
* Added cancellation token for Maintenance runtime processes
* Introduced generic data upload wizard
* Added Archive Migration UI component
* Added view angles to the Level 0 Product quality report
* Enabled support for encrypting and decrypting incremental backup files
* Allowed overriding of runtimeId in executor configuration
* Updated Subscription page to allow manual resource assignment
* Alphabetically sorted the subscription list dropdown
* Added support for generating executor metadata schema files
* Enabled Catalogue search by archiveId
* Added Create Hub Item resource functionality
* Supported versioning when version tag is not included in product ID
* Introduced shared executor service
* Added UI page for deleting subscriptions
* Created hierarchical event structure for application logging
* Added support for generating ad-hoc auxiliary files
* Integrated incremental backup management into the Backup UI
* Enforced immutability of resource IDs
* Add support for incremental backup sources
* Integrated Sentinel-2 Cloud Masks into Ancillary Mirror

### Bug Fixes
* Fixed authorisation errors that occur when editing and saving Hub Items
* Resolved issue preventing Geni product downloads for free-portal users
* Fixed layout break when large description text is used in the Create Hub Item modal
* Addressed display issues on the Generic Radiometric & Geometric Wizard Tutorial page
* Fixed resource assignment not saving for owning subscriptions on the Subscription page
* Resolved filesystem archive error in diagnostics workflow
* Fixed CSV parsing error in Satellite Store Scheduler
* Subscription page now correctly updates after delete actions
* Fixed synchronization issue affecting incremental backups for some order keys
* Improved handling of order persistence errors
* Ensured all data is listed correctly in the pickup browser
* Corrected execution of unintended Docker executors
* Fixed error encountered while restoring order database entries
* Resolved incomplete chuking issues experienced in the Gateway UI
* Added missing bean to restore standalone Gateway functionality
* Fixed incorrect subscription context used by Maintenance Executors for certain orders
* The appId tag is now properly ignored during workflow processing
* Resolved report failures caused by missing workflow assignment
* Fixed null navattPoints error in L1C report generation
* Handled fallback failure for solar angle data that caused code 400 errors
* Fixed update issue in Subscription Status View when shared resources are used
* Resolved archive conflict when multiple data stores used the same resource ID
* Fixed duplicate key error (NIR1) during report generation
* Moved app and info executors to the root applications page
* Fixed type erasure issue in STAC product file generation
* Product file now updates correctly during archive sync to reflect missing assets

### Support
* Adjusted the order of Save, Delete, and Cancel buttons for consistency
* Removed deprecated Archive Rename Executor
* Removed maintenance executors from the API executor list
* Updated logic for generating User Subscriptions
* Moved system properties from Job to JobAssignment or JobRecord
* Refactored maintenance workflows to be generated dynamically in code
* Reviewed and updated the relationship between the wizard and workflow components
